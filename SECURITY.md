# Security Policy

## Supported Versions

Only the latest stable version of QCAFlow is currently supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| < 5.1   | :x:                |

## Reporting a Vulnerability

We take the security of QCAFlow seriously. If you find a security vulnerability, please do not open a public issue. Instead, report it privately to the maintainer's contact email listed in `CITATION.cff` or repository metadata.

Once reported, we will:
1. Acknowledge receipt of the report within 48 hours.
2. Work on a fix and test it.
3. Publish a patched release and notify the reporter.
