# Project Roadmap

This roadmap outlines the planned future direction and milestones for the development of **QCAFlow**.

## Milestone 1: Core Algorithm Extensions (Planned Q4 2026)
*   **Coincidence Analysis (CNA):** Integrate an experimental implementation of the CNA algorithm as a configurational causal discovery alternative to traditional truth table minimization, supporting causal chain structures.
*   **ConCovOpt Integration:** Implement the consistency/coverage optimization protocol (ConCovOpt) to help researchers find optimal consistency and frequency thresholds mathematically rather than arbitrarily.

## Milestone 2: Technical & Portability Improvements (Planned Q1 2027)
*   **Progressive Web App (PWA):** Implement service workers to enable full offline execution and local installations on mobile and desktop systems without internet dependencies.
*   **mvQCA Support:** Extend the truth table and minimization logic to support Multi-Value QCA (mvQCA).

## Milestone 3: Open Science & Integrations (Planned Q2 2027)
*   **Repository Integrations:** Build options to export data and configuration directly to Open Science Framework (OSF) and Zenodo repositories.
*   **R Interoperability:** Export generated configurations directly as runnable R script code (compatible with packages `QCA` and `SetMethods`).
